# `core/` — 순수 로직 계층

> 의존 방향: `ui → core ← data` (ADR-0002 §3). **core는 아무것도 import 하지 않는다.**

## 역할
앱의 심장. 캐스케이드 우선순위 엔진과 상태 전이 로직을 프레임워크·저장소·시간에서 격리한 **순수 함수** 모음.

## 의존 규칙 (반드시 지킬 것)
- **import 금지**: React·Dexie·Capacitor·전역시간(`Date.now()`)·IO·랜덤 일절 없음.
- **시간 주입**: `now`를 인자로 받음(테스트가 시간을 고정).
- **결정성**: 같은 입력 → 같은 출력. 랜덤 타이브레이크 금지(`createdAt` 고정).
- 효과: 골든 테스트로 결정 재현성 증명(면접 자산) · v2 학습 엔진 교체점 · 플랫폼/저장소 교체 시 무손상.
